# Tinyhouse
A Full-stack Typescript React & GraphQL application.

## Technologies used:

### Client:
- React
- React Router DOM
- GraphQL with Apollo Client
- Ant Design
- Typescript

### Server:
- Node js
- Express
- GraphQL with Apollo Server
- Mongodb
- OAuth 2.0 with Google
- Google People's API
- Google Geocoding API
- Cloudinary
- Stripe
